
# Electric Vehicle Analysis Dashboard

## Overview
An interactive Power BI dashboard analyzing EV adoption trends, manufacturer performance, vehicle types, and CAFV eligibility using 150K+ electric vehicle records.

## Key Insights
- BEVs account for 78% of total vehicles
- EV adoption increased rapidly after 2020
- Tesla leads total EV registrations
- Model Y and Model 3 are the most popular EV models

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Features
- EV trends by model year
- State-wise analysis
- BEV vs PHEV comparison
- Manufacturer analysis
- Interactive dashboard filters
